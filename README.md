* Rails 6.0.5
# 就業Termの技術
---

* Ajaxを使ったコメント機能
# カリキュラム外の技術
---
* seed-fu

# 実行手順
---
$ git clone git@github.com:kimura-kaori/one.git  
$ cd one  
$ bundle install  
$ rails db:create && rails db:migrate  
$ rails s
